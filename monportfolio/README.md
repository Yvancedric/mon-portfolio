# Portfolio Frontend

Frontend React pour le portfolio avec design moderne, animations douces et responsive.

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Le serveur de développement sera accessible sur `http://localhost:3000`

## Build

```bash
npm run build
```

## Fonctionnalités

- ✅ Design system CSS avec variables
- ✅ Animations douces avec Framer Motion
- ✅ Navigation intuitive (moins de 3 clics)
- ✅ Multilingue (FR/EN)
- ✅ Responsive design
- ✅ Accessibilité (WCAG 2.1 AA)
- ✅ SEO optimisé
- ✅ Feedback utilisateur (chargements, actions)

## Structure

```
src/
├── components/     # Composants réutilisables
├── contexts/       # Contextes React (Language)
├── pages/          # Pages principales
├── services/       # Services API
├── App.jsx         # Composant principal
├── main.jsx        # Point d'entrée
└── index.css       # Design system CSS
```

## Technologies

- React 18
- React Router
- Framer Motion (animations)
- Lucide React (icônes)
- Axios (API)
- React Helmet Async (SEO)
